# PATANJALI AROGYA KENDRA Billing System

## Introduction

The **PATANJALI AROGYA KENDRA Billing System** is a console-based Python application developed to simplify the billing process for a Patanjali store. It allows users to place orders using product codes, calculates the total bill, generates an order summary, and keeps track of daily sales.

## Features

* Displays a list of available products with their codes and prices.
* Accepts product codes and quantities from the user.
* Calculates the total bill automatically.
* Generates an order summary with date and time.
* Maintains daily sales records.
* Displays the final sales summary before exiting.
* Handles invalid product codes and incorrect quantity inputs.

## Technologies Used

* Python 3
* `datetime` module

## How to Run the Project

1. Install Python 3 on your system.
2. Save the project file as `products.py`.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python products.py
```

## Working

1. The program displays the available products with their prices.
2. The user enters the product code and quantity.
3. Multiple products can be added to the same order.
4. The application calculates the total amount.
5. An order summary with the current date and time is displayed.
6. The daily sales are updated after every order.
7. When the user exits, the application displays the total sales summary for the day.
